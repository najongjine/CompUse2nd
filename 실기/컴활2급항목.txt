컴퓨터활용능력 2급 실기 기본 항목 + 세부 출제 항목

표시 기준
- [시험 빈출]: 실제 실기 문제에서 반복적으로 나오는 핵심 항목
- 표시가 없는 항목: 출제 범위에는 포함되지만 우선순위는 상대적으로 낮은 항목

1. 참조
- 상대 참조와 절대 참조
  - 상대 참조: 수식을 복사하면 행/열 주소가 같이 바뀌는 참조
  - [시험 빈출] 절대 참조: $A$1처럼 수식 복사 시 고정되는 참조
  - [시험 빈출] 혼합 참조: $A1, A$1처럼 열 또는 행만 고정하는 참조
  - [시험 빈출] 채우기 핸들로 수식 복사하기
  - [시험 빈출] 계산 범위와 조건 범위가 어긋나지 않게 지정하기
  - [시험 빈출] 함수식에서 기준표, 할인율표, 등급표 등을 절대 참조로 고정하기

2. 함수
- 논리 함수
  - [시험 빈출] IF: 조건에 따라 다른 값 표시
  - IFS: 여러 조건을 순서대로 판단
  - [시험 빈출] IFERROR: 오류가 나면 지정한 값으로 대체
  - [시험 빈출] AND, OR, NOT: 여러 조건을 함께 판단
  - TRUE, FALSE: 논리값 사용
  - SWITCH: 값에 따라 결과를 분기
  - [시험 빈출] 자주 나오는 유형: 합격/불합격, 등급, 수수료, 할인 여부, 조건별 문구 표시

- 날짜와 시간 함수
  - [시험 빈출] TODAY, NOW: 현재 날짜/현재 날짜와 시간
  - [시험 빈출] YEAR, MONTH, DAY: 날짜에서 연/월/일 추출
  - HOUR, MINUTE, SECOND: 시간에서 시/분/초 추출
  - [시험 빈출] DATE, TIME: 날짜나 시간 만들기
  - [시험 빈출] WEEKDAY: 요일 번호 구하기
  - [시험 빈출] DAYS: 두 날짜 사이 일수 계산
  - WORKDAY: 주말을 제외한 근무일 기준 날짜 계산
  - EDATE, EOMONTH: 몇 개월 후 날짜, 월말 날짜 계산
  - [시험 빈출] 자주 나오는 유형: 나이, 근속기간, 만기일, 요일, 납기일, 월말일 계산

- 문자열 함수
  - [시험 빈출] LEFT, RIGHT, MID: 왼쪽/오른쪽/중간 문자 추출
  - [시험 빈출] LEN: 글자 수 구하기
  - [시험 빈출] FIND, SEARCH: 특정 문자의 위치 찾기
  - TRIM: 불필요한 공백 제거
  - UPPER, LOWER, PROPER: 대문자/소문자/첫 글자 대문자 변환
  - [시험 빈출] 자주 나오는 유형: 주민번호, 상품코드, 부서코드, 지역코드에서 필요한 문자 추출

- 수학과 삼각 함수
  - [시험 빈출] SUM: 합계
  - [시험 빈출] SUMIF, SUMIFS: 조건에 맞는 합계
  - [시험 빈출] ROUND, ROUNDUP, ROUNDDOWN: 반올림/올림/내림
  - [시험 빈출] INT, TRUNC: 정수 부분 구하기
  - [시험 빈출] MOD: 나머지 구하기
  - ABS: 절댓값
  - POWER: 거듭제곱
  - RAND, RANDBETWEEN: 난수 발생
  - [시험 빈출] 자주 나오는 유형: 총액, 할인액, 세금, 단가 계산, 특정 자리 반올림, 짝수/홀수 판정

- 데이터베이스 함수
  - [시험 빈출] DSUM: 조건에 맞는 합계
  - [시험 빈출] DAVERAGE: 조건에 맞는 평균
  - [시험 빈출] DCOUNT: 조건에 맞는 숫자 데이터 개수
  - [시험 빈출] DCOUNTA: 조건에 맞는 비어 있지 않은 데이터 개수
  - [시험 빈출] DMAX, DMIN: 조건에 맞는 최대값/최소값
  - [시험 빈출] 자주 나오는 유형: 조건표를 만들어 부서별/지역별/품목별 합계, 평균, 개수 구하기
  - [시험 빈출] 주의할 점: 데이터베이스 범위에는 필드명이 포함되어야 하고, 조건 범위의 필드명은 원본과 같아야 함

- 찾기와 참조 함수
  - [시험 빈출] VLOOKUP: 세로 방향 표에서 값 찾기
  - [시험 빈출] HLOOKUP: 가로 방향 표에서 값 찾기
  - [시험 빈출] INDEX: 지정한 행/열 위치의 값 가져오기
  - [시험 빈출] MATCH: 찾는 값의 위치 구하기
  - CHOOSE: 번호에 따라 값 선택
  - ROW, ROWS: 행 번호/행 개수
  - COLUMN, COLUMNS: 열 번호/열 개수
  - [시험 빈출] 자주 나오는 유형: 코드표에서 품명, 단가, 등급, 할인율, 부서명 찾기
  - [시험 빈출] 주의할 점: 정확히 일치(FALSE)와 근사값(TRUE) 구분, 기준표 범위 절대 참조

- 통계 함수
  - [시험 빈출] AVERAGE, AVERAGEA: 평균
  - [시험 빈출] AVERAGEIF, AVERAGEIFS: 조건에 맞는 평균
  - [시험 빈출] COUNT, COUNTA, COUNTBLANK: 숫자 개수/비어 있지 않은 셀/빈 셀 개수
  - [시험 빈출] COUNTIF, COUNTIFS: 조건에 맞는 개수
  - [시험 빈출] MAX, MAXA, MIN, MINA: 최대값/최소값
  - [시험 빈출] LARGE, SMALL: n번째 큰 값/작은 값
  - [시험 빈출] RANK.EQ: 순위
  - MEDIAN: 중앙값
  - MODE.SNGL: 최빈값
  - STDEV.S, VAR.S: 표본 표준편차/표본 분산
  - [시험 빈출] 자주 나오는 유형: 순위, 상위 몇 명, 조건별 인원수, 평균 이상/미만 판정

- 함수 25문제
  - [시험 빈출] 기본 계산 문제: 사칙연산, 셀 참조, 백분율 계산
  - [시험 빈출] 조건 계산 문제: IF, AND, OR, COUNTIF, SUMIF 조합
  - [시험 빈출] 날짜/문자 추출 문제: 날짜 함수와 문자열 함수 조합
  - [시험 빈출] 참조표 활용 문제: VLOOKUP, HLOOKUP, INDEX, MATCH
  - [시험 빈출] 통계 문제: 순위, 개수, 평균, 최대값, 최소값
  - [시험 빈출] 출제 포인트: 함수 하나만 쓰는 문제보다 여러 함수를 중첩하는 문제가 자주 나옴

3. 기본 작업 / 분석 작업
- 셀 서식
  - [시험 빈출] 글꼴, 크기, 굵게, 기울임, 밑줄, 색상
  - [시험 빈출] 셀 병합, 가운데 맞춤, 가로/세로 정렬
  - [시험 빈출] 테두리, 채우기 색, 무늬
  - [시험 빈출] 표시 형식: 숫자, 통화, 회계, 백분율, 날짜, 시간, 사용자 지정
  - [시험 빈출] 사용자 지정 표시 형식: "개", "원", 0, #, 쉼표, 음수 표시, 조건별 색 표시
  - 행 높이, 열 너비, 자동 맞춤
  - 페이지 설정: 여백, 방향, 인쇄 영역, 반복할 행/열, 머리글/바닥글

- 고급 필터
  - [시험 빈출] 조건 범위 작성
  - [시험 빈출] 같은 행 조건: AND 조건
  - [시험 빈출] 다른 행 조건: OR 조건
  - [시험 빈출] 비교 조건: >=, <=, <>, 와일드카드(*, ?)
  - [시험 빈출] 계산 조건: 수식 조건 사용
  - [시험 빈출] 필터 결과를 다른 위치에 복사
  - 고유 레코드만 추출
  - [시험 빈출] 주의할 점: 조건 범위의 필드명, 복사 위치의 필드명, 원본 데이터 범위 확인

- 조건부 서식
  - [시험 빈출] 특정 값보다 크거나 작은 셀 강조
  - 상위/하위 값 강조
  - 중복 값, 고유 값 표시
  - [시험 빈출] 수식을 사용하여 서식 지정
  - [시험 빈출] 행 전체에 서식 적용하기
  - 데이터 막대, 색조, 아이콘 집합
  - [시험 빈출] 자주 나오는 유형: 평균 이상, 특정 부서, 특정 문자 포함, 조건을 만족하는 행 색칠

- 텍스트 나누기
  - [시험 빈출] 구분 기호로 나누기: 쉼표, 공백, 하이픈, 탭
  - 고정 너비로 나누기
  - [시험 빈출] 열 데이터 형식 지정: 일반, 텍스트, 날짜
  - [시험 빈출] 주민번호, 상품코드, 주소, 이름/전화번호 분리
  - 주의할 점: 기존 데이터 덮어쓰기 여부 확인

- 정렬
  - [시험 빈출] 오름차순, 내림차순
  - [시험 빈출] 여러 기준으로 정렬
  - 사용자 지정 목록 정렬
  - 행 단위 정렬과 열 단위 정렬
  - 색 기준, 아이콘 기준 정렬
  - [시험 빈출] 주의할 점: 표 전체 범위를 함께 선택해야 행 데이터가 섞이지 않음

- 부분합
  - [시험 빈출] 기준 필드로 먼저 정렬
  - [시험 빈출] 그룹별 합계, 평균, 개수, 최대값, 최소값
  - 여러 단계 부분합
  - 부분합 윤곽 기호 사용
  - 부분합 제거
  - [시험 빈출] 자주 나오는 유형: 부서별 합계, 지역별 평균, 품목별 판매량 집계

- 목표값 찾기
  - [시험 빈출] 수식 셀, 목표값, 값을 바꿀 셀 지정
  - [시험 빈출] 원하는 결과가 나오도록 입력값 역산
  - 자주 나오는 유형: 목표 매출, 목표 평균, 목표 점수, 목표 이익률 맞추기
  - [시험 빈출] 주의할 점: 수식 셀에는 반드시 계산식이 있어야 함

- 데이터 표
  - 단일 변수 데이터 표
  - 이중 변수 데이터 표
  - 행 입력 셀, 열 입력 셀 지정
  - 여러 조건에 따른 결과 비교
  - 자주 나오는 유형: 이자율/기간별 상환금, 판매량/단가별 매출, 할인율별 금액

- 통합
  - [시험 빈출] 여러 범위의 데이터를 하나로 집계
  - [시험 빈출] 합계, 평균, 개수, 최대값, 최소값 통합
  - [시험 빈출] 첫 행/왼쪽 열 레이블 사용
  - 원본 데이터와 연결
  - [시험 빈출] 자주 나오는 유형: 지점별, 월별, 부서별 자료를 하나의 표로 통합

- 시나리오
  - 변경 셀 지정
  - 시나리오 추가, 편집, 삭제
  - 시나리오 요약 보고서 작성
  - 여러 상황의 결과 비교
  - 자주 나오는 유형: 낙관/보통/비관 상황별 매출, 비용, 이익 비교

- 피벗테이블
  - [시험 빈출] 행/열/값/필터 영역 배치
  - [시험 빈출] 값 필드 요약 방식 변경: 합계, 평균, 개수 등
  - [시험 빈출] 표시 형식 변경
  - [시험 빈출] 그룹화: 날짜, 숫자 범위
  - [시험 빈출] 필터, 정렬, 슬라이서
  - 피벗 차트와 연동
  - [시험 빈출] 자주 나오는 유형: 부서별 합계, 월별 판매량, 지역별 평균, 담당자별 실적 분석

4. 기타 작업
- 매크로
  - [시험 빈출] 매크로 기록
  - [시험 빈출] 매크로 이름 지정
  - 바로 가기 키 지정
  - 상대 참조로 기록하기
  - [시험 빈출] 도형/단추에 매크로 연결
  - [시험 빈출] 매크로 실행 및 삭제
  - [시험 빈출] 자주 나오는 유형: 서식 지정 매크로, 인쇄 설정 매크로, 특정 영역 이동/복사 매크로
  - [시험 빈출] 주의할 점: 시험에서는 복잡한 VBA 작성보다 기록과 실행이 중심

- 차트
  - [시험 빈출] 차트 종류 선택: 세로 막대형, 꺾은선형, 원형, 혼합형 등
  - [시험 빈출] 차트 제목, 축 제목, 범례, 데이터 레이블 설정
  - [시험 빈출] 축 서식: 최소값, 최대값, 표시 단위, 눈금
  - [시험 빈출] 데이터 계열 서식: 색상, 표식, 선, 보조 축
  - [시험 빈출] 차트 스타일과 레이아웃 변경
  - [시험 빈출] 차트 위치 이동: 새 시트 또는 기존 시트
  - [시험 빈출] 데이터 범위 수정, 행/열 전환
  - [시험 빈출] 자주 나오는 유형: 매출 추이, 항목별 비율, 목표 대비 실적, 두 자료를 보조 축으로 비교
